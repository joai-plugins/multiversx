# MultiversX

Connect MultiversX to Claude, Codex, and ChatGPT. Check balances, view transactions, manage staking, and interact with ESDT tokens through natural language.

[Open app page](https://joai.ai/en/apps/multiversx) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-multiversx@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-multiversx --url https://cortex.joai.ai/mcp/apps/multiversx
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/multiversx.git ~/.cursor/plugins/local/joai-multiversx
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-multiversx&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktbXVsdGl2ZXJzeCI6eyJ1cmwiOiJodHRwczovL2NvcnRleC5qb2FpLmFpL21jcC9hcHBzL211bHRpdmVyc3gifX19) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/multiversx
```
